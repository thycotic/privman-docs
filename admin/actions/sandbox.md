# Sandbox Action

This type of action will limit the environments in which certain code can execute. The sandbox runs a process in a job objectÂ that limits its ability to interact with other processes, as well as limiting some specific types of interactions with the operating system.

![Display Sandbox Settings](images/sandbox.png "Display Sandbox Settings")

## Parameter

The following Sandbox Action Settings can be enabled:

* Limit Desktop
* Limit Global Atoms
* Limit Display Settings
* Limit System Parameters
* Limit Write Clipboard
* Limit Handles
* Limit Exit Windows
* Limit Read Clipboard
